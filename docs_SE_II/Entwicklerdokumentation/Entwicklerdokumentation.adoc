=== Verwendete Bibliotheken/Frameworks...

Wir verwenden als Framework verwenden wir Django, welches auf der Programmiersprache Python basiert, da wir das Projekt der vorangehenden Grupppe als Grundlage nutzen. Wir nehmen die Version 3 oder höher, da es sonst zu Kompatibilitätsproblemen mit verschiedenen Funktionen von Django selbst kommt. Zusätzlich werden folgende Erweiterungen von Python und Django benötigt:

* mator
* gunicorn
* psycopg2
* django-money
